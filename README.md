# Expense Tracker

Personal expense tracker built with vanilla JavaScript, HTML, and Tailwind CSS — installable on mobile as a PWA.

🔗 **Live demo:** https://taupe-sable-e1fef6.netlify.app

## About
Built as a portfolio mini project — learning to build a web app from scratch (manual state management, DOM manipulation, localStorage) before moving to frameworks.

## Features
- Add and delete expense transactions
- Auto-calculate total expenses
- Data persists after refresh (localStorage)
- Works offline (PWA + service worker)
- Installable to iPhone/Android home screen

## Tech Stack
- HTML, vanilla JavaScript
- Tailwind CSS
- PWA (manifest + service worker)
- Deployed on Netlify
